## Kenney Tower Defense (Top-Down)

- **Source**: https://kenney.nl/assets/tower-defense-top-down
- **Author**: Kenney (kenney.nl)
- **License**: CC0 1.0 Universal (Public Domain)
- **Description**: 299 top-down tower defense tiles (64x64) including terrain (grass, dirt, sand, stone), path pieces, tower bases, turrets, enemy sprites (bugs, airplanes), projectiles (rockets, arrows, fire effects), and UI elements (numbers, coins). Uniform grid tilesheet.
