# Asset Attribution

## Kenney Space Shooter Redux

- **Source**: https://kenney.nl/assets/space-shooter-redux
- **Author**: Kenney (kenney.nl)
- **License**: CC0 1.0 Universal (Public Domain)
- **Description**: Space shooter sprites including 3 player ship designs (4 colors + damage states), 5 enemy ship types (4 colors), UFOs, lasers, meteors, power-ups, engine effects, shields, and UI elements. Single combined sprite sheet with XML atlas metadata.
