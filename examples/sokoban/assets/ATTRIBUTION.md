# Asset Attribution

## Kenney Sokoban

- **Source**: https://kenney.nl/assets/sokoban
- **Author**: Kenney (kenney.nl)
- **License**: CC0 1.0 Universal (Public Domain)
- **Description**: Sokoban puzzle game sprites including colored crates (brown, blue, green, red with normal and solved states), wall tiles, floor tiles (brick, stone, grass, dirt), target markers, player character sprites (4 directions with walk animation frames), and coins. Both uniform tilesheet (64x64 grid) and XML atlas spritesheet included.

## Planned SFX (to be sourced separately)

- `move.ogg` — Player moves to adjacent cell
- `push.ogg` — Player pushes a crate
- `solved.ogg` — Crate placed on target
- `undo.ogg` — Undo last move
- `level-complete.ogg` — All targets covered
- `cant-move.ogg` — Invalid move attempted
